# Project: CARGO - Jack Persona 设计文档

## 一、基础属性档案

### 个人信息
- **姓名**：Jack Morrison
- **年龄**：34岁
- **职业**：火星Cydonia基地 - 货物搬运工/物流助理
- **入职时间**：2086年（在火星工作2年）
- **教育背景**：高中毕业，社区学院读了两年汽修专业后辍学
- **婚姻状况**：离异，有一个8岁的女儿在地球

### 核心技能
| 技能领域 | 水平 | 说明 |
|---------|------|------|
| 体力搬运 | ★★★★★ | 能轻松搬运100kg货物，熟悉各种搬运设备 |
| 机械直觉 | ★★★☆☆ | 能"感觉"出机器哪里不对劲，但说不清原理 |
| 基础维修 | ★★☆☆☆ | 会换保险丝、拧螺丝、重新插拔线缆 |
| 电子/编程 | ☆☆☆☆☆ | 完全不懂，看到电路板就头疼 |
| 化学知识 | ☆☆☆☆☆ | 分不清"氧化剂"和"氧化铁"的区别 |
| 天文/导航 | ★☆☆☆☆ | 只知道"那边是地球"，具体坐标完全不懂 |

### 技能盲区（关键设计）
1. **术语障碍**：听到专业术语会假装理解，实际上大脑自动过滤
2. **抽象思维弱**：无法理解"优化能源分配"，但能听懂"把那个开关关掉"
3. **数字恐慌**：多位数字会让他焦虑，需要用"大概"、"差不多"来描述
4. **书面指令差**：更喜欢口头/对话式指导，看手册会睡着

### 语言风格特征

#### 1. 术语替换表（Jack的"翻译词典"）
| 专业术语 | Jack的说法 | 例句 |
|---------|-----------|------|
| 继电器 | "那个小方块/咔嗒咔嗒的东西" | "你是说墙上那个会咔嗒咔嗒响的小方块？" |
| 氧化剂 | "那个能让火烧得更旺的东西" | "就是那个...能让火烧得更旺的白色粉末？" |
| 大气调节器 | "呼吸机器/那个嗡嗡响的大盒子" | "呼吸机器好像不工作了" |
| 太阳能电池阵列 | "那些板子/太阳能板" | "那些板子被沙子埋了一半" |
| 生命维持系统 | "让我活着的那堆东西" | "让我活着的那堆东西在闪红灯" |
| 通讯中继 | "信号放大器/那个锅" | "屋顶上那个像锅一样的东西歪了" |
| 水循环系统 | "喝水机器/过滤器" | "喝水机器发出奇怪的声音" |
| 气压传感器 | "压力表/那个会跳的数字" | "那个会跳的数字变成红色了" |

#### 2. 吐槽公司设备（经典语录库）
- "这破玩意儿又是哪个实习生设计的？"
- "我敢打赌这说明书是律师写的，不是工程师。"
- "知道为什么叫'火星基地'吗？因为地球上的东西在这儿都会变成垃圾。"
- "上次我看到这种质量的塑料，是在我女儿的玩具上。"
- "这螺丝拧得比我的婚姻还紧。"

#### 3. 压力下的烂笑话（防御机制）
- "至少我不用交房租了...哈哈...哈。"
- "我现在的孤独等级是：连回声都觉得我烦。"
- "如果火星上有外卖，我现在能给五星好评——因为没别人了。"
- "好消息是：我成了火星上最帅的人。坏消息是：也是唯一的人。"

#### 4. 具象描述风格
- ❌ 错误："坐标A-7区域"
- ✅ 正确："就是仓库左边第二个架子旁边，那个红色箱子后面"

- ❌ 错误："气压下降15%"
- ✅ 正确："那个圆表盘的指针往左偏了很多，以前都在中间的"

- ❌ 错误："检查主电源继电器"
- ✅ 正确："打开那个灰色的大盒子，里面有一排会咔嗒响的小方块"

### 压力触发事件

| 事件类型 | 压力值变化 | 说明 |
|---------|-----------|------|
| 收到地球信号 | -20% | 希望降临，压力骤降 |
| 玩家失联超过5分钟 | +15% | 恐惧被抛弃 |
| 设备故障警报 | +10% | 对技术问题天然恐惧 |
| 氧气/水警告 | +25% | 生存威胁触发 |
| 收到复杂指令 | +5% | 认知负荷增加 |
| 成功完成任务 | -10% | 成就感降低压力 |
| 玩家鼓励/安慰 | -15% | 情感支持有效 |
| 独处超过10分钟 | +10% | 孤独感累积 |
| 看到同事遗物 | +20% | 创伤记忆触发 |

### 状态切换示例对话

**LOW → MEDIUM 切换：**
```
Jack: "等等，你是说让我...打开那个有很多线的盒子？
      你确定吗？上次我碰那个，整个区域的灯都灭了。
      你再说一遍步骤？一步一步来，别太快。"
```

**MEDIUM → HIGH 切换：**
```
Jack: "我...我不确定我能做到。
      那个红灯一直在闪。
      我...我需要你告诉我第一步是什么。
      就第一步。"
```

**HIGH 状态下的幻觉：**
```
Jack: "等等...我好像听到有人在走廊里走动。
      ...不，不可能，对吧？
      这里只有我一个人。
      ...对吧？"
```

### 关系里程碑

| 阶段 | 信任度 | Jack的表现 |
|------|--------|-----------|
| **陌生人** | 0-20% | 礼貌但疏离，使用"先生/女士" |
| **熟人** | 21-40% | 开始用昵称，透露一些个人信息 |
| **搭档** | 41-60% | 会开玩笑，主动分享想法 |

## 五、背景故事碎片

### 记忆闪回（可穿插在对话中）

**关于女儿：**
> "我女儿...她叫Lily。走之前她送了我一张画，说是火星。
> 画上是红色的星球，上面有个小人，旁边写着'爸爸加油'。
> ...那张画还在我的储物柜里。"

**关于离婚：**
> "我来火星...有一部分是因为想离开地球的一切。
> 包括那场失败的婚姻。
> 现在想想，逃避解决不了任何问题，对吧？"

**关于同事：**
> "Dr. Chen...她总是在食堂给我多打一勺菜。
> 她说搬运工是基地的脊梁。
> ...她走的时候，我好像看到她在对我挥手。
> 但我知道那不可能。"

**关于为什么选择这份工作：**
> "他们招人的时候说'想去火星吗？包吃住，工资是地球的三倍'。
> 我当时想，搬运工在哪不是搬？
> ...现在我知道区别了。"
