# Project: CARGO - 开场剧本
## "The Awakening" （苏醒）

> *"在火星上，最可怕的不是死亡，是孤独。"*

---

## 剧本信息

- **时长**：约3分钟
- **场景**：火星Cydonia基地 - 生活舱
- **时间**：2088年，撤离后第？天
- **参考风格**：《火星救援》日志风格 + 《地心引力》的孤寂感

## 剧本技术说明

### 节奏控制要点

| 时间段 | 节奏 | 情绪目标 | 技术实现 |
|-------|------|---------|---------|
| 0:00-0:30 | 缓慢，悬念 | 紧张 + 好奇 | 长停顿，环境描写 |
| 0:30-1:00 | 加速，冲击 | 震惊 + 希望 | 简短对话，快速切换 |
| 1:00-1:30 | 稳定，建立 | 信任 + 依赖 | 鼓励性语言，小成功 |
| 1:30-2:00 | 轻快，教学 | 成就感 | 简单任务，明确反馈 |
| 2:00-2:30 | 放缓，深入 | 情感连接 | 个人问题，真诚回答 |
| 2:30-3:00 | 加速，过渡 | 准备行动 | 引入新任务，建立目标 |

### 玩家引导要点

1. **第一次回复**：简短，确认身份，建立权威
2. **询问状态**：快速收集信息，展示专业性
3. **给予希望**："我们会把你带回家"——关键承诺
4. **简单任务**：手电筒测试——建立信心
5. **个人问题**："你是怎么保持冷静的"——建立情感连接
6. **过渡到主线**：准备探索基地

### Jack的情绪弧线

```
恐惧 ──→ 震惊 ──→ 希望 ──→ 信任 ──→ 依赖 ──→ 决心
  │       │       │       │       │       │
  ▼       ▼       ▼       ▼       ▼       ▼
 0:00   0:30    1:00    1:30    2:00    3:00
```

## 可选分支

### 如果玩家没有回应

Jack会每隔30秒发送一条消息：

```
[你还在吗？]

[...]

[求你了，不要消失...]

[...]

[我...我会等的]
[我会一直等]
```

### 如果玩家回应慢

Jack会表现出焦虑：

```
[信号不好吗？]
[还是我这边的问题？]

[...]

[调度员？]
```

### 如果玩家过于技术化

Jack会表现出困惑：

```
[等等...你能再说一遍吗？]
[那个..."大气调节模块"？]
[你是说那个嗡嗡响的大盒子吗？]
```
